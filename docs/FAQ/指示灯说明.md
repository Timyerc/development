# 指示灯说明

## 圆形
|序号|机器状态|指示灯状态|说明|
|:---|:---|:---|:---|
|1|机器上电后|红灯常亮|机器未吸附|
|2|--|红灯闪5次|陀螺仪异常|
|2|--|红灯闪7次|2.4G异常|
|3|--|红灯闪8次|小单片机异常|
|4|--|红灯闪9次|气压异常|
|5|--|红灯闪10次|涂鸦模块异常|
|6|--|黄灯常亮|机器充电中|
|7|--|绿灯常亮|机器充电完成|
|8|机器行走|红灯闪1次|机器出现短暂漏气|
|9|--|红灯频闪|适配器断开|
|10|--|蓝灯闪烁|机器处于自动喷水模式|
|11|--|蓝灯常亮|机器处于手动喷水模式|
