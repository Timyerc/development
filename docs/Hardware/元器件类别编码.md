# 元器件类别编码

## odoo元器件编码规则

编码由类别和随机码组成，中间用短横线隔开：

[类别]-[随机码]

类别为两位数字编码，对应关系如下：

| 编码 | 元器件类别 |
| ---- | --------- |
| 00 | PCB |
| 01 | 芯片, 传感器 |
| 02 | 电容 |
| 03 | 电阻 |
| 04 | 电感 |
| 05 | 二极管, LED |
| 06 | 三极管, 场效应管 |
| 07 | 连接器, 连接线 |
| 08 | 开关, 按键 |
| 09 | 显示屏 |
| 10 | 胶件 |
| 11 | 硅胶件 |
| 12 | EVA |
| 13 | 螺丝 |
| 14 | 天线 |
| 15 | 磁铁 |
| 16 | 蜂鸣器, 扬声器, 咪头 |
| 17 | 金属件 |
| 18 | 电池 |
| 19 | 螺丝刀 |
| 20 | 包装，说明书 |
| 21 | 马达 |
| 22 | PCBA |
| 23 | PCBA套件 |
| 24 | 软件授权 |

随机码为5位随机数字编码

例子：`01-05530`
